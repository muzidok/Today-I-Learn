### 15.animate()

```js
$('선택자').animate({CSS 스타일}, 지속시간, 이징(easein), 콜백함수);
```

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>
    <style>
      .box {
        width: 100px;
        height: 100px;
        background-color: rgb(249, 184, 4);
      }
    </style>
  </head>
  <body>
   <button class="btn">click</button>
   <div class="box"></div>
    </div>
  </body>
  <script>
 $('.btn').click(function() {
    $('.box').animate({
        width: '100%',
        borderRadius: '30px'
    }, 5000 ,function(){
        $(this).css('background-color', 'red');
    });
 })
  </script>
</html>

```

#### animate() -delay()

애니메이션 지연설정 fadeIn(), slideDown()과 같이 쓰인다.

```html
  <head>
    <script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>
    <style>
      .box {
        width: 100px;
        height: 100px;
        background-color: rgb(249, 184, 4);
        margin: 10px
      }
    </style>
  </head>
  <body>
<button class="btn">click</button>
   <div class="box box1"></div>
   <div class="box box2"></div>
   <div class="box box3"></div>
    </div>
  </body>
  <script>
 $('.btn').click(function() {
    $('.box1').delay(0).animate({width:'100%'},1000);
 })

 $('.btn').click(function() {
    $('.box2').delay(500).animate({width:'100%'},1000);
 })

 $('.btn').click(function() {
    $('.box3').delay(1000).animate({width:'100%'},1000);
 })
  </script>
```

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>
    <style>
      .move {
        width: 100px;
        height: 100px;
        background-color: rgb(249, 184, 4);
        margin: 10px;
        display: flex;
        justify-content: center;
        align-items: center;
        position: relative;
      }
    </style>
  </head>
  <body>
    <div class="di">
      <button class="box box1">우측이동</button>
      <button class="box box2">좌측이등</button>
    </div>
    <div class="move">MOVE</div>
  </body>
  <script>
    $(".box1").click(function () {
      $(".move").delay(500).animate({ left: "+=100" }); // 버튼을 누를때마다 100px씩 상대이동
    });
    $(".box2").click(function () {
      $(".move").delay(500).animate({ left: "-=100" });
    });
  </script>
</html>
```

## 100글자시 경고문 띄우기

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>
    <style>
      textarea {
        width: 100px;
        height: 100px;
        padding: 20px;
        outline: none;
      }
      .msg1 {
        color: red;
        display: none;
      }
    </style>
  </head>
  <body>
    <div class="di">
      <textarea id="t1" maxlength="100"></textarea>
      <div class="di2">
        <p class="msg1">입력수가 초과되었다.</p>
        <p class="count"><span class="count2">0</span>/100</p>
      </div>
    </div>
  </body>
  <script>
    $("#t1").keydown(function () {
      let n = $(this).val().length;
      $(".count2").text(n);
      n >= 100
        ? $(".msg1").show().css({ border: "3px solid blue" })
        : $(".msg1").hide();
    });
  </script>
</html>
```
